# BibliEco
This is a repository created for a project the BiblioEco aimed at waste pickers.

## Getting Started
First, run the development server:

```bash
npm install
#or
npm install -y

npm run dev
# or
yarn dev
# or
pnpm dev
```

After is open in the url http://localhost:3000 to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
